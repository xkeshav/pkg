# alphabet

A simple npm package which return array of english alphabets ( length of 26 )

## How to install

```sh
> npm install @xkeshav/alphabet
```

## Usage

```tsx
import { alphabet } from '@xkeshav/alphabet';

console.log({ alphabet }); // return ['A', 'B', ... 'Z']
```
